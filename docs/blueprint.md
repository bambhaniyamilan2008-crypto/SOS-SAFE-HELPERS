# **App Name**: SafeHelp

## Core Features:

- SOS Alert Triggering: Allows users to instantly activate an SOS alert via a prominent on-screen button, a shake gesture (if device supported), or voice command.
- Emergency Contact Management: Users can add, edit, and prioritize a list of contacts to be notified during an emergency, including their relationship.
- Real-time Location Sharing: Automatically shares the user's current location with designated emergency contacts upon SOS activation.
- Voice-Activated SOS: An AI-powered voice recognition system enables hands-free activation of SOS alerts using predefined keywords for enhanced accessibility.
- Customizable Alert Messaging: Users can pre-configure a personalized message that will be sent automatically with their SOS alert to provide crucial information.
- Accessible UI Controls: Provides essential accessibility settings such as large text, dark mode, and voice guidance to accommodate diverse user needs.
- Secure Medical Info Storage: Allows users to securely store vital medical information, such as blood type and disability type, for quick retrieval in emergencies.

## Style Guidelines:

- Color Anchor: Urgent Assistance & Critical Information. Scheme: Dark. Primary color (for alert/action): A vibrant, alert-focused orange-red (#FA6723). Hue 10, Saturation 95%, Lightness 60%.
- Background color: A very deep, warm charcoal grey (#201B1A). This hue (10) is derived from the primary, heavily desaturated (20%) and dark (10%).
- Accent color: A bright, luminous magenta-pink (#E36EF7) to draw attention to secondary important elements. This analogous hue (340) is derived from the primary (10), saturated (80%) and lighter (70%) to create strong contrast.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, techy, and urgent feel. Body text font: 'Inter' (sans-serif) for high readability, especially for accessibility-focused content.
- Use custom, bold, and high-contrast icons that feature a glowing or backlit appearance, drawing visual inspiration from the SOS symbol and microphone found in the reference design.
- Minimalist, high-contrast layout emphasizing large, clearly delineated interactive elements. Ample negative space will reduce visual clutter, enhance focus, and improve accessibility.
- Implement subtle glow effects on interactive components, smooth transitions between screens, and clear visual feedback for actions such as button presses or SOS activation, including a dynamic countdown.
